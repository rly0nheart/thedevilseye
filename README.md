# Installation
## Github
```
pip install git+https://github.com/rly0nheart/thedevilseye
```
## PyPI
```
pip install thedevilseye
```
# Usage
```
thedevilseye <query>
```
### Note
> Make sure you have selenium geckodriver properly setup
>> Firefox browser needs to be installed on system
